# Images Folder

This folder should contain image assets used in the app.

## Required Images

- `sportverse_icon.png` - Main app icon/logo (referenced in login and OTP screens)
  - Recommended size: 200x200px or larger
  - Format: PNG with transparency

## Usage

Images in this folder are automatically included when referenced in code:
```dart
Image.asset('assets/images/sportverse_icon.png')
```

## Adding Images

1. Place your image files in this directory
2. The images will be bundled with the app automatically
3. Make sure to use appropriate image sizes to optimize app size
